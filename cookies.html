Çerez Politikası

Sweet Bonanza Game olarak, web sitemizi ziyaret ettiğinizde kullanıcı deneyiminizi geliştirmek için çerezler kullanıyoruz. Bu çerez politikası, çerezlerin nasıl kullanıldığını ve bunları nasıl yönetebileceğinizi açıklar.

1. Çerez Nedir?

Çerezler, bir web sitesini ziyaret ettiğinizde cihazınıza indirilen küçük metin dosyalarıdır. Bu dosyalar, web sitesi üzerinde gezinirken tercihlerinizi hatırlamak gibi işlevler sağlar.

2. Kullandığımız Çerez Türleri

  • Zorunlu Çerezler: Web sitemizin düzgün çalışması için gerekli olan çerezlerdir.
  • Performans Çerezleri: Web sitemizin nasıl kullanıldığını analiz etmek için kullanılır ve kullanıcı deneyimini geliştirmemize yardımcı olur.
  • Fonksiyonel Çerezler: Tercihlerinizi (dil, bölge vb.) hatırlamak için kullanılır.
  • Hedefleme/ Reklam Çerezleri: İlgi alanlarınıza uygun reklamlar sunmamıza olanak tanır.

3. Çerezlerin Kullanım Amaçları

Çerezler, aşağıdaki amaçlar için kullanılabilir:

  • Kullanıcı oturumlarını yönetmek
  • Web sitesinin performansını artırmak
  • Kullanıcı tercihlerini ve ayarlarını hatırlamak
  • Kişiselleştirilmiş içerik ve reklam sunmak

4. Çerezlerin Yönetimi

Çerezleri tarayıcı ayarlarınızdan kontrol edebilir veya silebilirsiniz. Çoğu tarayıcı, çerezleri otomatik olarak kabul eder, ancak bu ayarı değiştirebilirsiniz. Çerezleri devre dışı bırakmayı seçerseniz, web sitemizin bazı özellikleri düzgün çalışmayabilir.

5. Üçüncü Taraf Çerezleri

Web sitemizde, üçüncü taraf hizmet sağlayıcıları tarafından yerleştirilen çerezler kullanılabilir. Bu çerezler, üçüncü tarafların kendi gizlilik politikalarına tabidir.

6. Çerez Politikası Değişiklikleri

Bu çerez politikası zaman zaman güncellenebilir. Herhangi bir değişiklik yapıldığında, bu sayfada güncellenmiş politika yayınlanacaktır.

7. İletişim

Çerez politikamızla ilgili sorularınız için lütfen bizimle iletişime geçin.
